FROM employees_clean
GROUP BY Department
ORDER BY attrition_rate DESC
💡
Key Finding
Contrary to the initial hypothesis, compensation is not the primary driver of attrition — employees who left earned only $101/month less than those who stayed. The dominant signal is overtime status, which increases attrition risk by 52%, followed by low job satisfaction and stalled promotions. This reframes the retention strategy from compensation-focused to workload and career development-focused.

Flight Risk Model

Composite 6-factor score · 0–10 scale
A weighted composite score was engineered to rank employee segments by attrition likelihood. Each factor reflects a statistically significant driver identified in the pipeline analysis. Score 0 = minimal risk, 10 = critical risk.
Python feature_engineering.py — FlightRiskScore
# Composite weighted risk score — higher = more likely to leave
df['FlightRiskScore'] = (
  (df['OverTime'] == 'Yes').astype(int) * 2.5      # Strongest single signal
  + (df['JobSatisfaction'] <= 2).astype(int) * 2.0  # Low engagement
  + (df['MonthlyIncome'] < 3000).astype(int) * 2.0  # Below market
  + (df['WorkLifeBalance'] <= 2).astype(int) * 1.5  # Burnout signal
  + (df['YearsSinceLastPromotion'] > 5).astype(int) * 1.5 # Career stall
  + (df['StockOptionLevel'] == 0).astype(int) * 0.5  # No retention equity
)

About This Project

This project demonstrates an end-to-end data pipeline workflow applied to a real-world HR analytics problem. It mirrors the structure of a production data pipeline: ingest, audit, clean, transform, aggregate, model, and present — each stage documented and reproducible.

The dataset (IBM HR Analytics style, 1,470 records) includes intentional data quality issues — nulls, duplicates, and mixed types — to demonstrate realistic cleaning judgment. All transformations use explicit, defensible strategies (median imputation for skewed income, mode for satisfaction scores).

The Flight Risk Score is an original feature engineering contribution: a weighted composite of six behavioral signals empirically linked to attrition. The scoring logic is designed to be directly translatable to SQL for deployment in a BI tool or HR dashboard.

The headline insight — that overtime is a stronger attrition driver than compensation — has direct strategic implications: retention investment should prioritize workload management and promotion cadence over pay adjustments alone.
