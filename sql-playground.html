SQL Query Playground

A live, in-browser SQL environment preloaded with a realistic retail operations dataset. Write your own queries or run the examples to explore joins, aggregations, window functions, and more.

// Database Schema — RetailCo Operations Dataset (4 tables, 500+ rows)

employees

  • emp_id INTEGER PK
  • name TEXT
  • department TEXT
  • region TEXT
  • hire_date TEXT
  • salary REAL
  • status TEXT

sales

  • sale_id INTEGER PK
  • emp_id INTEGER FK
  • product_id INTEGER FK
  • sale_date TEXT
  • quantity INTEGER
  • revenue REAL
  • region TEXT

products

  • product_id INTEGER PK
  • product_name TEXT
  • category TEXT
  • unit_price REAL
  • cost REAL

performance

  • review_id INTEGER PK
  • emp_id INTEGER FK
  • review_year INTEGER
  • score REAL
  • rating TEXT

Note

This playground runs entirely in your browser using sql.js (SQLite compiled to WebAssembly). No data is sent to any server. The dataset is synthetically generated to represent a realistic retail operations scenario with employees across four regions, product sales from 2021 to 2024, and annual performance reviews.
