Methodology

Excel-native analysis pipeline from raw enrollment data to visualization-ready tables

01
Distance Lookup
Each in-person student's home Census tract (GEOID) was matched to pre-computed campus distances. Online students received 0 miles by design.
IF([@Campus]="WEB", 0,
SUMIFS(Raw_Distances[Distance],
  Raw_Distances[GEOID], [@GEOID],
  Raw_Distances[Campus],
  VLOOKUP([@Campus], Campus_Key, 2, FALSE)))
02
Distance Binning
Continuous distances were grouped into 20-mile bands using a ROUNDDOWN formula, enabling pivot-ready categorical analysis.
IF([@Distance_km]=0, "0-19",
IF([@Distance_km]>300, ">300",
ROUNDDOWN([@Distance_km]/20,0)*20 & "-" &
(ROUNDDOWN([@Distance_km]/20,0)*20+19)))
03
GEOID Quality Flag
Every student record was classified with a data quality status — flagging blank GEOIDs, unmatched tracts, and online-exempt records.
IF([@Campus]="WEB","WEB-no distance",
IF([@GEOID]="","Blank GEOID",
IF(ISNA(VLOOKUP([@GEOID],
  Raw_Distances[[GEOID]:[Distance]],1,FALSE)),
"GEOID not found","OK")))
04
Pivot Table Design
Two pivot tables were built: one showing % of grand total by campus × distance band, and one profiling GEOID match quality across the dataset.
Pivot_Distance: Rows=Distance_Bin
Columns=Campus_Full
Values=Count of STUDENT_ID
Show Values As: % of Grand Total
05
Cumulative Calculation
Per-campus cumulative percentages were computed as running sums divided by each campus's total share — isolating the 80% crossing point per campus independently.
Cumml_East = SUM($B$2:B2) / SUM($B$2:$B$17)
Cumml_Main = SUM($C$2:C2) / SUM($C$2:$C$17)

About This Work

This project demonstrates a complete business intelligence workflow in Excel — from raw enrollment records to a clean, presentation-ready analysis. It was designed to answer a real institutional question about geographic recruitment reach.

The analysis connects 18,407 student records to 15,788 Census tract–to–campus distance pairs using GEOID spatial identifiers, employing SUMIFS, VLOOKUP, and nested IF logic to classify and measure each student's commute band. Pivot tables, cumulative columns, and chart-ready data sheets were all built from scratch without any external tools.

Key technical decisions include separating raw data from analytical outputs across dedicated sheets, implementing explicit data quality flags before running any aggregation, and using ROUNDDOWN-based binning for reproducible categorization of continuous distance values.
