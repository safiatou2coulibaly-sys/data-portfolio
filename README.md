## Project 02 — Employee Attrition & Flight Risk Pipeline

End-to-end data pipeline analysis on 1,470 employee records — from raw dirty data to a composite flight risk model. Built to answer: what actually drives people to leave? Pipeline stages: ingest, audit, clean, impute, feature engineering, aggregation, and a weighted 6-factor flight risk scoring model. Key finding: overtime is a stronger attrition driver than compensation, increasing attrition risk by 52%. Built with Python/pandas logic and SQL-equivalent queries.

**Live page:** https://safiatou2coulibaly-sys.github.io/data-portfolio/attrition.html
